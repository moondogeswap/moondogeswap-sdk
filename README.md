## Running tests

To run the tests, follow these steps. You must have at least node v10 and [yarn](https://yarnpkg.com/) installed.

First clone the repository:

```sh
git clone https://github.com/moondogeswap/moondogeswap-sdk.git
```

Move into the moondogeswap-sdk working directory

```sh
cd moondogeswap-sdk/
```

Install dependencies

```sh
yarn install
```

Run tests

```sh
yarn test
```
